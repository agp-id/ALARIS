## ALARIS (Arch Linux Artix Runit Installer Script)

### Thanks to:
- Allah SWT


### Usage:

 Install:
```sh
  sh -c "$(curl -L https://raw.githubusercontent.com/agp-id/ALARIS/main/installer.sh)"
```
 Check/edit & run:
```sh
  curl -O https://raw.githubusercontent.com/agp-id/ALARIS/main/installer.sh
  nano installer.sh
  sh installer.sh
```
